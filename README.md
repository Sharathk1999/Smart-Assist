# Smart Assist - AI Chat Application

Smart Assist is an AI-powered chat application utilizing the Google Gemini API to provide a seamless conversational experience with diverse functionalities.

![Smart Assist](path/to/your/gif_or_image.gif)

## Features

### Normal Chat
Engage in text-based conversations with the AI, enabling a wide range of discussions and queries.

### Chat with Image and Text
Exchange both images and text seamlessly within the chat interface, allowing for rich communication.

### Text Chat
Standard text-based chatting functionality, facilitating instant communication with the AI.

### Embed Content Chat
Share and interact with embedded content such as links, videos, or other media formats during conversations.

## Getting Started

To run Smart Assist locally, follow these steps:

1. Clone the repository: `git clone https://github.com/your_username/SmartAssist.git`
2. Navigate to the project directory: `cd SmartAssist`
3. Install dependencies: `npm install` or `yarn install`
4. Set up the required API keys for Google Gemini in the configuration file.
5. Run the application: `npm start` or `yarn start`

## Usage

1. Open the Smart Assist application.
2. Begin by initiating a conversation or sending a query.
3. Explore different features like text, image, or embedded content chatting.
4. Enjoy the interactive experience provided by Smart Assist!
